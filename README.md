# T.T.DEVELOPPEMENT — Site vitrine

Site vitrine sur-mesure pour T.T.DEVELOPPEMENT, agence web indépendante.
Sites livrés en 48h pour 800 €.

## Stack
HTML5 + CSS (custom properties) + JavaScript vanilla. Aucune dépendance.

## Structure
- `index.html` — page unique
- `styles.css` — styles (tokens via variables CSS)
- `script.js` — menu mobile, validation du formulaire, anti-spam
- `robots.txt` / `sitemap.xml` — SEO

## Développement local
Ouvrir `index.html` dans un navigateur, ou servir le dossier :
```bash
python3 -m http.server
```

## Déploiement
Site statique — déployable tel quel sur Vercel, Netlify ou tout hébergeur statique.
